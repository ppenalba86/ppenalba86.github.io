Proyecto final de Distribuidora

Para la creación de la base de datos "PROYECTO_FINAL_DISTRIBUIDORA" se debe compilar en orden los siguientes archivos: 1 Creacion_de_Tablas: En este archivo se crea el esquema llamado "PROYECTO_FINAL_DISTRIBUIDORA" y las tablas correspondientes a esta. 2 Insercion_de_Datos: Este archivo contiene el script a insertar los datos para las tablas. 3 Vistas: Este Script contiene las sentencias donde se pueden apreciar las vistas. 4 Funciones : Este script contiene las sentencias de las funciones realizadas. 5 Store_Procedures: Contiene los procesos almacenados. 6 Triggers + Tablas Log: Este Script contiene los disparadores para las tablas provedores y producto. 7 Transactions: Script que contiene dos sentencias de Luenguaje de Control Transaccional. 8 Sentencias: Script que contiene la creación de dos usuarios. 9 Backup : Archivo Sql que contiene copia de seguridad de los datos de las tablas pedido de compra, pedido de venta y producto.
